## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This is the official repository for engaato online shoe e-commerce app
The app uses firebase for the backend and vercel for the front end
After cloning the repo, modify the root directory with .env.local file containing the api key and credntials from firebase
